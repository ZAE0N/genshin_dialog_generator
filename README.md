# genshin_dialog_generator

원신(Genshin Impact)·붕괴: 스타레일(Honkai: Star Rail) 스타일 대화창 이미지를
만드는 웹 편집기입니다. 상단 드롭다운에서 게임을 선택하면 해당 게임의 대화 UI로
렌더링됩니다. 포토샵 없이 브라우저에서 이름·대사·선택지를 입력하고 PNG로 저장할 수 있습니다.

## 사용법

`index.html`을 브라우저에서 열면 끝. 설치·빌드·서버가 필요 없습니다.

1. **게임 선택** — 상단에서 원신 / 스타레일을 고릅니다. (전환 시 기본 이름·예시 선택지가 게임에 맞게 바뀝니다)
2. **이미지 업로드** — 캔버스로 쓸 배경/캐릭터 이미지를 올립니다. 우상단 ✕ 로 지울 수 있어요.
3. 이미지의 가로·세로 비율을 읽어 **대화창과 선택지가 자동 배치**됩니다.
   (832×1216 같은 세로형, 1920×1080 같은 가로형 모두 지원 · 위치는 하단 기준 픽셀 고정)
4. **이름 / 대사 / 선택지**를 입력합니다. 대사는 최대 5줄까지 자동 줄바꿈됩니다.
5. **PNG로 저장** — 업로드한 이미지의 원본 해상도 그대로 렌더링됩니다. (미리보기 = 저장 결과)

### 게임별 특징
- **원신**: 중앙 이름(+부제)과 마름모 장식 구분선, 우측 끝에 밀착된 말풍선 선택지.
  선택 표시(●) 시 노란 배경/노란 글씨 토글 지원.
- **스타레일**: 크림골드 이름과 중앙 장식 구분선, 검은 외곽선 글자, 우측 선택지.
  선택지마다 아이콘 타입(진행 →, 잡담 💬, 임무 ‹!›, 종료 ⇥, 조사 🔍 등)을 고를 수 있고,
  선택 시 밝은 막대로 강조됩니다.

### 폰트
원신·스타레일 인게임 폰트(HanYi HYWenHei 계열)는 상용이라 내장하지 않고, 기본은
**Noto Sans KR**(온라인 시)로 근사합니다. **폰트 파일 불러오기**(.ttf/.otf/.woff/.woff2)로
직접 가진 폰트를 적용할 수 있으며, 폰트는 브라우저 안에서만 쓰이고 어디에도 업로드되지 않습니다.

## 리소스 출처

- `assets/genshin/talk/`, `assets/hsr/talk/` — [PathOfGenshin/resources](https://github.com/PathOfGenshin/resources)에서
  추출한 인게임 UI 스프라이트 (자세한 내용은 각 폴더의 README 참고)
- 배치 기준 — 실제 인게임 스크린샷(원신·스타레일)을 픽셀 단위로 대조하여 위치·색·크기를 맞춤.
  원신은 [Kaerralind의 Genshin Dialogue Template](https://www.deviantart.com/kaerralind/art/Genshin-Dialogue-Template-922592432)
  (무료 사용, 크레딧 불요 명시)도 참고.
- 폰트 — 상용 폰트라 내장하지 않고 Noto Sans KR로 근사 (위 폰트 섹션 참고).
